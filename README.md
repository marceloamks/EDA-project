# EDA-project

📈 E-commerce Sales Forecast

📌 Overview

This project analyzes e-commerce sales data and predicts future revenue using machine learning models.

---

📊 Key Insights

«💡 Sales have shown strong growth over time
👉 Indicates increasing demand»

«💡 Sales increase towards weekends
👉 Focus promotions on these days»

«💡 Forecast shows continued growth
👉 Prepare inventory and marketing»

---

🤖 Models Used

- Linear Regression (baseline)
- Prophet (advanced time series model)

---

📈 Results

- Forecasted revenue shows continued growth (~1.04M → 1.10M)
- Seasonal patterns impact sales performance
- Weekly and yearly trends influence revenue

---

🚀 Business Impact

This analysis can help:

- Improve sales strategy
- Optimize marketing timing
- Increase revenue

---

🛠️ Tools

- Python
- Pandas
- Matplotlib
- Prophet
